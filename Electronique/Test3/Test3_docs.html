Présentation du projet

Ce projet a pour but de concevoir un afficheur 7 segments mécanique innovant basé sur 7 servomoteurs, chacun représentant un segment. L'objectif est d'afficher les chiffres de 0 à 9 puis de 9 à 0 de manière séquentielle sans utiliser de composants lumineux.

Contraintes techniques

  • Utiliser 7 servomoteurs (un par segment)
  • Utiliser un microcontrôleur Atmega328P (sans carte Arduino)
  • Alimentation par batterie lithium
  • Pas de fonction bloquante comme delay()
  • Pas d’utilisation de breadboard ni de carte Arduino dans le montage final
  • Utilisation possible du module PCA9685 pour la gestion des servos
